# Operators in C

##  Overview

Operators are special symbols that perform operations on variables and values.

They are one of the fundamental building blocks of C programming and are used in almost every program. Understanding operators is essential for problem-solving, embedded systems programming, and technical interviews.

- Equal to (`==`)
- Not Equal (`!=`)

## 4. Assignment Operators

- Assignment (`=`)

# Data Types in C

## Overview

Data types define the type of data a variable can store and determine the memory allocated for that variable.

Understanding data types is essential for writing efficient C programs, especially in Embedded Systems where memory is limited.

---

## Topics Covered

1. int
2. char
3. float
4. double
5. short
6. long
7. signed
8. unsigned
9. sizeof()
10. Type Casting

---

## Learning Outcomes

After completing this module, you will be able to:

- Understand all basic C data types
- Choose the correct data type for a program
- Optimize memory usage
- Use type casting correctly
- Understand signed and unsigned integers
- Apply data types in Embedded C projects
